# IEMS

        本项目是后端基于Spring+SpringMVC+Mybatis，前端基于Layui开发的一款方便用户查看和操作的可视化进销存管理系统，能够实现对进销存数据的增删查改等基础功能，支持数据分页，模糊搜索等功能，同时利用Echarts进行数据可视化处理，生成对应图表。

        本项目功能不完整，仅作为SSM架构与Echarts学习的小型项目，有待后续完善与更新。
